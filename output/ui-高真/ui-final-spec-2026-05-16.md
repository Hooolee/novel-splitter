# Novel Splitter UI 定稿（2026-05-16）

参考基线：
- `output/ui-高真/library_v2.png`
- `output/ui-高真/novel-detail_v1.png`

本次定稿目标：把其余页面统一到同一套“暖米纸感 + 铜橙主色 + 轻玻璃纸面容器”的视觉语言，并将前端落地到可交付状态。

## 一、定稿覆盖范围

1. 全局应用外壳（顶栏 / 内容容器 / 底栏）
2. 书库页列表态
3. 书库卡片态 / 空状态
4. 报告详情阅读态
5. 表单控件基础风格（按钮 / 输入框 / 筛选 chips）

> 注：已有高真已覆盖书库主界面和书籍详情页，本轮将这些页面的实际前端也往同一视觉方向统一。

## 二、视觉语言

### 1) 色彩
- 主背景：`#F6EFE4`
- 纸面渐变：`rgba(255,250,243,0.96) -> rgba(247,239,228,0.92)`
- 主文字：`#4A3123`
- 次级文字：`#8F7564`
- 主强调色：`#C96C3A`
- 主按钮渐变：`#D77A46 -> #C96535`
- 边框主色：`#DEC8B0`
- Hover 背景：`#F1E3D2`
- Active 背景：`#F9ECD6`

### 2) 圆角
- 顶层 shell：28px
- 一级面板：22px ~ 24px
- 按钮：14px
- 标签 / 筛选：999px

### 3) 阴影
- 顶层容器：`0 18px 48px rgba(137,93,56,0.12)`
- 卡片：`0 6px 18px rgba(137,93,56,0.10)`
- 主按钮：`0 10px 20px rgba(201,108,58,0.26)`

### 4) 字体层级
- App 标题：24px / 600
- Tab 文案：20px / 500
- 卡片书名：20px / 600
- 面板小标题：12px~13px / 600~700
- 正文：14px~15px / 400~500
- 说明辅助：11px~12px / 400

## 三、页面级定稿说明

### A. 全局壳层
- 头部改为纸面壳容器，统一与高真一致的温暖留白。
- Logo 强化为铜橙色徽章，增加产品识别。
- 中央双 tab 改成更接近 mockup 的大尺寸 segmented control。
- 设置 / 主题切换按钮改为圆角图标按钮，和纸面边框统一。

### B. 书库页列表态
- 顶部操作区：工作目录按钮改为纸面按钮；“添加书籍”为主 CTA；“刷新”为次按钮。
- 筛选条 / 标签区：收纳为面板式容器，视觉噪音更低。
- 列表卡片整体更贴近高真右侧书卡：
  - 平台与上榜信息先行
  - 书名更大
  - 评级 badge 用柔和状态底色
  - 底部 CTA 做成纸面按钮

### C. 空状态
- 从“系统占位文案”改为可交付空态：
  - 更大图标
  - 标题“书库尚未点亮”
  - 明确引导添加书籍或先去扫榜

### D. 报告详情阅读态
- 报告阅读页整体包在纸面大壳中。
- TOC 左栏单独成列，改用更轻的 hover 与更清晰的标题层级。
- Markdown 阅读区切换为浅底深字，更贴近文稿 / 分析报告氛围。

## 四、状态规范

### 共识标签
- 一致看好：边框 `#9BB47D` / 背景 `#F1F6E8` / 文本 `#68834D`
- 多数看好：边框 `#B3C68E` / 背景 `#F5F8ED` / 文本 `#779255`
- 分歧：边框 `#EFC48F` / 背景 `#FFF6EA` / 文本 `#D27F33`
- 多数不看好：边框 `#EFB0A1` / 背景 `#FFF1EE` / 文本 `#D8654F`
- 一致不看好：边框 `#E79D92` / 背景 `#FFF0ED` / 文本 `#C95144`

### 交互态
- hover 位移：`translateY(-1px ~ -2px)`
- hover 阴影增强，但不使用强蓝/强黑
- 选中卡片：主边框 + 轻 ring 高亮

## 五、验收清单

- [ ] 头部、书库列表、报告阅读页都统一到同一暖米色体系
- [ ] 主要按钮层级清晰：主按钮铜橙，次按钮纸面
- [ ] 卡片可读性高于旧版，书名与评估结论更醒目
- [ ] 空状态不再像调试页面，而像正式产品页面
- [ ] 构建通过：`npm run build`
